# VUETIFY

### Vue 기반 UI 프레임워크

구글의 Material Design 기반으로 설계됨

---

### 프레임워크란

frame + work 로 틀을 갖고 일한다.

ex) 눈오리 → 많은 눈오리 → 눈오리 틀(프레임 워크)이 필요함

---

### 장점

빠르게 만들수 있다.

일관성 있다.

퀄리티 있게 만들수 있다.

---

### UI 프레임 워크

생산성이 높다.

일관성이 있어 유지보수가 용이하다.

지식이 부족해도 퀄리티 있게 만들수 있다. (UI/UX 지식)

코드의 재사용성이 높다.
